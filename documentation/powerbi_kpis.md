# KPI Power BI recommandés

## 1. Spend Total
- Définition : somme totale des montants d'achat
- Usage : carte KPI ou graphique en colonnes
- Objectif : suivre la masse totale dépensée

## 2. Savings Amount
- Définition : montant total des économies réalisées
- Usage : indicateur de performance financière

## 3. Savings %
- Définition : $\frac{\text{savings amount}}{\text{spend total}} \times 100$
- Usage : KPI de performance de réduction des coûts

## 4. OTD %
- Définition : pourcentage de livraisons à temps
- Usage : benchmark fournisseur et suivi logistique

## 5. Quality %
- Définition : taux de qualité moyen observé
- Usage : suivi qualité fournisseur

## 6. RSE Score
- Définition : score moyen de performance RSE / ESG
- Usage : analyse fournisseur responsable

## 7. Top 10% Spend Concentration
- Définition : part du spend concentrée dans les 10 % des fournisseurs les plus gros acheteurs
- Usage : vue de concentration des achats

## 8. Supplier Count
- Définition : nombre de fournisseurs actifs
- Usage : segmentation et comparaison de portefeuille
